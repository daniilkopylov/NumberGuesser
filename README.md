# Консольная игра «Угадай число» (Java)

Программа представляет собой интерактивную игру, в которой компьютер загадывает случайное число от 1 до 100, а у пользователя есть 7 попыток на его угадывание. Архитектура ввода построена на низкоуровневом посимвольном чтении `System.in.read()`, что гарантирует защиту от сбоев буфера при некорректном вводе. Ошибочный ввод (буквы, числа вне диапазона 1-100) не засчитывается как потраченная попытка.

## 🧠 Ключевые особенности игрового алгоритма

1. **Математический баланс (Бинарный поиск):** Лимит в 7 попыток выбран неслучайно. По законам теории алгоритмов, диапазон из 100 значений гарантированно сужается до 1 конкретного числа за $\log_2(100) \approx 6.64$ шагов, если на каждом этапе отсекать ровно половину вариантов с помощью подсказок «больше/меньше».
2. **Иммунитет к потере попыток:** Блок инкремента `attemptsUsed++` расположен строго *после* матрицы валидации входных данных. Если пользователь ошибается в формате (вводит текст или отрицательное число), игра выводит предупреждение и возвращает его на шаг ввода, сохраняя текущий счетчик попыток нетронутым.
3. **Безопасность типов данных:** Предварительный сбор данных в тип `long` страхует арифметику от умышленного или случайного ввода сверхбольших чисел, которые могли бы вызвать арифметическое переполнение знакового бита `int`.
